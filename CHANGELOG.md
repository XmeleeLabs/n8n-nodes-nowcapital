# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-01-28
### Added
- **Initial Public Release** of the NowCapital n8n node.
- Operation: **Calculate Sustainable Spend** for retirement optimization.
- Operation: **Get Detailed Projections** for year-by-year financial breakdowns.
- Operation: **Calculate Specific Spend** for modeling target cash flows.
- Operation: **Monte Carlo Simulation** (Premium) for risk analysis.
- **Smart Polling**: Automatic sub-task detection and "handover" following for asynchronous Monte Carlo simulations.
- **Resource Pattern**: Organized operations under the "Plan" resource for better n8n UI consistency.
- **Linting & Best Practices**: Fully compliant with n8n community node standards, including sorted properties and strict typing.
- **Connection Testing**: Built-in "Test Connection" functionality in credentials.
- **Branding**: Official NowCapital logo for both nodes and credentials.
- **Documentation**: Comprehensive README with polling loop guides and async execution examples.
